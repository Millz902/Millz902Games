# Sound Files for Los Scotia Police System

This directory should contain the following sound files in OGG format:

## Required Sound Files:

1. **alert.ogg** - General alert notification sound
2. **panic.ogg** - Emergency panic button sound  
3. **dispatch.ogg** - Dispatch radio sound
4. **siren.ogg** - Police siren sound
5. **radio_beep.ogg** - Radio communication beep

## Usage:
These sounds are played during various police activities:
- Alert notifications for incoming calls
- Panic button activation
- Dispatch communications
- Vehicle siren activation
- Radio communications

## Format Requirements:
- File format: OGG Vorbis
- Sample rate: 44.1 kHz recommended
- Bit depth: 16-bit minimum
- Mono or stereo acceptable

## Installation:
Place the OGG sound files directly in this directory. The resource will automatically load them when the police system is used.

## Note:
Due to copyright considerations, actual sound files are not included. Server administrators should source appropriate royalty-free or licensed police/emergency sounds and convert them to OGG format.
